Exercise 2.1 - Bootstrap Directory Structure

1. CSS Folder
Purpose:
Contains Bootstrap stylesheets used to design web pages.

Important Files:
- bootstrap.css : Full Bootstrap CSS file.
- bootstrap.min.css : Minified version for production use.
- bootstrap-grid.css : Grid system only.
- bootstrap-utilities.css : Utility classes only.

2. JS Folder
Purpose:
Contains JavaScript files required for interactive Bootstrap components.

Important Files:
- bootstrap.js : Full Bootstrap JavaScript.
- bootstrap.bundle.js : Includes Popper.js.
- bootstrap.min.js : Minified version.
- bootstrap.bundle.min.js : Minified bundle with Popper.js.

3. Icons Folder (if downloaded separately)
Purpose:
Contains Bootstrap Icons used to add graphical symbols to web pages.

Examples:
- house icon
- search icon
- person icon
- heart icon

Conclusion:
The CSS folder provides styling, the JS folder provides interactive functionality, and the Icons folder provides scalable vector icons for UI design.
